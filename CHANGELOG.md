# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-02-16

### Added
- Initial release of OpenClaw Smart Task Manager
- Smart timeout management (30 minutes with auto-compaction)
- Model failover support between providers
- Task decomposition strategy
- Context pruning with cache-ttl mode
- Concurrency control (2 main tasks, 4 subtasks)
- Heartbeat monitoring (every 30 minutes)
- Dual-instance support (main + backup)
- Status monitoring script
- Comprehensive documentation in English and Chinese
- Task breakdown prompt templates

### Features
- Automatic configuration optimization
- Progress tracking to PROGRESS.md
- Session reset strategies
- Error handling and logging
- Best practices from Moltbook community

### Documentation
- Complete README with bilingual support
- Detailed usage guide
- Configuration examples
- Troubleshooting section
- Best practices references
